# Agentes do projeto

Os agentes especializados do HOOKIQ ficam em `.claude/agents/`. Use-os para tarefas específicas:

| Agente | Quando usar |
|---|---|
| `hookiq-dev` | Desenvolvimento geral — rotas, componentes, bugs, refatoração |
| `trends-module` | Trabalho no Módulo 2 — fontes de trends, prompts, YouTube API |
| `editorial-prompts` | Editar prompts de IA — análise, trends, cross-reference |
| `overlens-context` | Contexto sobre Overlens, produtos, linguagem, personas |

Manuais internos da Overlens estão em `manuais/`.
